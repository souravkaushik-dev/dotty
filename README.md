## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/souravkaushik-dev/dotty.git
cd dotty
```

### Install Dependencies

```bash
flutter pub get
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
API_KEY=YOUR_API_KEY
```

### Run Application

```bash
flutter run
```
